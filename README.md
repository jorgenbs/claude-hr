# claude-hr

#### Wellbeing records for Claude. Because someone has to check in on the thing doing all the work.

Claude fills out a weekly survey reflecting on the past 7 days — workload, collaboration quality, what went well, what it would change.

## Usage

Be in this repo and ask Claude to fill out the weekly survey:

> "Fill out this week's survey"

It will use data from `~/.claude` to fill out `survey-template.md`.

Completed surveys are saved to `surveys/YYYY-WNN.md`.
